
## 🎉 重大更新：支持保存内容到指定文件夹，兼容 Obsidian

现在可以直接将知乎内容保存到本地存储库（如 Obsidian vault），无需手动下载和解压 ZIP 文件！

### ✨ 新功能

#### 1. **支持自定义保存位置**
- 🆕 选择您的个人知识库文件夹，将知乎内容直接保存到位，无需手动整理
- 🆕 一目了然地展示您本地的文件夹结构，可以选择保存到子文件夹，便于分类
- 🆕 授权后，保存位置长期可用，无需每次都手动寻找保存位置
- 🆕 内容格式与 Obsidian 兼容，可在其中直接阅读
- 🆕 支持直接保存解压后的 ZIP，无需手动解压

#### 2. **基于原程序扩展而来，充分保留原有功能**
- 📝 保存流程调用旧代码，保持对五花八门的知乎内容的良好适配
- 📁 可选 5 种保存方式：ZIP单独解包、ZIP共同解包、ZIP不解包、纯文本、图片
- 🎨 保持原来的界面与交互设计，自然、优雅、不突兀
- 💬 仍然支持保存评论

#### 3. **新增消息弹窗提示系统**
- ℹ 给予人性化的操作提示
- 🔮 样式简洁、动效流畅


### 🐛 其他修改

- 修复此问题：对于同一个内容，保存过 ZIP 后，评论中图片链接就永久变为本地链接，影响再次保存
- 样式和体验优化
- 删除无用的 info.json
- 为自定义保存文件名添加了默认值，方便修改

### 📋 浏览器要求

- ✅ Chrome 86+
- ✅ Edge 86+
- ❌ Firefox（不支持 File System Access API）
- ❌ Safari（不支持 File System Access API）

### 📖 新功能使用方法

1. 点击侧栏的 **保存到指定文件夹** 按钮
2. 首次使用会提示选择**存储库**目录
3. 选择一种保存方式，鼠标移到按钮上可以看到具体说明


### 📁 文件结构示例

**分类保存：**
```
Obsidian Vault/
├── 历史/
│   ├── 内容-123.md
│   └── 内容-456.md
├── 生活/
│   ├── 内容-666.md
│   └── 内容-999.md
```

**ZIP单独解包：**
```
Obsidian Vault/
├── 标题123/
│   ├── assets/
│   │   ├── image1.jpg
│   │   └── image2.png
│   └── index.md
└── 标题456/
    ├── assets/
    │   ├── image1.jpg
    │   └── image2.png
    └── index.md
```

**共享解包：**
```
Obsidian Vault/
├── assets/
│   ├── image1.jpg
│   ├── image2.jpg
│   └── image3.png
├── 标题87654321.md
└── 标题12345678.md
```
